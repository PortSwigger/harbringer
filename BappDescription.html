HARbringer imports HTTP Archive (HAR) files directly into the site map while preserving original HTTP requests and responses. HARbringer maintains the exact captured traffic for authentic analysis of previously recorded sessions.

Features

  • Import complete HAR files with original request/response data preserved
  • Browse and select individual entries from a sortable table
  • Batch import all entries or import selected entries individually
  • Handle binary data, base64 encoding, and various character encodings
  • Support HAR files from major browsers and development tools

Usage

  1. Navigate to the "HARbringer" tab
  2. Click "Browse" to select a HAR file from your system
  3. Click "Load HAR" to parse and display entries in the table
  4. Select a specific entry and click "Send to Site Map" to import individually, or click "Import All to Site Map" to import all entries
  5. View imported traffic in the site map for analysis with other tools
